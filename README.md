# Welcome to Tasking

## URL Docs
The basic URL format is simple. Index.html (the home page) will be called when there is no URL ('/') or when it is called ('/Index.html')

The URL that calls the tasks will be '/task_file/tasks'

To append the database use '/task_file/task_add'

we can always change the URLs, but this is it for now.


I'll work on ways to hide the localhost address soon.
